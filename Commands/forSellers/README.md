Here is a concise bulleted list of the C500 Builder commands for Discord, as defined in your documentation:
 * /c500 setup - Launches the initial onboarding wizard to connect Stripe Express and Twitch accounts.
 * /c500 drop - Opens a pop-up modal window to create and post a new product listing embed.
 * /c500 live [Order ID] - Verifies the builder is currently live on Twitch to unlock escrowed funds.
 * /c500 ship [Order ID] [Tracking Number] - Marks an order as shipped, sends tracking to the buyer, and releases funds.
 * /c500 dashboard - Displays a private overview of active listings, pending orders, and escrow balance.
 * /c500 vouch @user - Gives a small boost to a buyer's Reputation Score.
 * /c500 help - Shows a quick reference list of available commands.
